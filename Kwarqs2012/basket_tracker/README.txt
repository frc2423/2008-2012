
Requirements:

    - Python 3.2
    - SIP 
    - RobotPy source code present at C:\WindRiver\workspace\robotpy

Build instructions:

    - Open VxWorks development shell 
        - Start Menu -> Wind River -> VxWorks 6.3 .. -> VxWorks Development Shell
    - Navigate to source code directory
    - Run 'run_sip.bat' to generate the SIP source code
    - Run 'make' to actually build the python module
    
    
Robot Installation instructions:

    - Run 'install.bat'
